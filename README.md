# rewards-challenge

The online assessment for contractor role at Charter Communications

## Usage

### Install dependencies

```
npm run install:dependencies
```

### Run both frontend and backend

```
npm run dev
```

### Run frontend only

```
npm run client
```

### Run server only

```
npm run server
```

### Run frontend unit test

```
npm run test:client
```

### Run backend unit test

```
npm run test:server
```
